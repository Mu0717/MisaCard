# 🚀 快速使用指南

## 新功能速览

### 1️⃣ 批量复制卡密

**在哪里：** 卡片列表 → 批量操作栏

**怎么用：**
```
1. 勾选卡片 ✅
2. 点击"批量复制卡密"按钮 🔵
3. 粘贴使用 📋
```

**复制格式：**
```
mio-xxxxx-xxxx-xxxx
mio-xxxxx-xxxx-xxxx
mio-xxxxx-xxxx-xxxx
```

---

### 2️⃣ 根据额度筛选

**在哪里：** 卡片列表 → 筛选栏

**怎么用：**
```
1. 在"筛选额度"输入框输入额度（如：1, 2, 5）
2. 点击"搜索"按钮 🔵
3. 查看筛选后的卡片列表 📊
4. 可进行批量操作 📋
```

---

## 🎯 快捷操作

| 操作 | 位置 | 快捷方式 |

| 批量复制卡密 | 卡片列表 | 选中 → 批量复制卡密 |
| 批量复制卡号 | 卡片列表 | 选中 → 批量复制 |
| 筛选额度 | 卡片列表 | 输入额度 → 搜索 |
| 组合筛选 | 卡片列表 | 多个条件 → 搜索 |
| 清除筛选 | 卡片列表 | 清空条件 → 搜索 |

## 💡 使用技巧

### 技巧1：快速选择卡片
点击"一键选中已激活未使用"按钮，快速选择所有已激活且未使用的卡片。

### 技巧2：组合筛选
1. 在卡片列表输入额度筛选
2. 同时选择状态、退款状态、使用状态
3. 点击搜索，精确定位目标卡片
4. 对筛选结果进行批量操作

### 技巧3：批量备份
定期使用"批量复制卡密"功能，将所有卡密备份到文本文件。

---

## ⚠️ 注意事项

1. **批量复制卡密** vs **批量复制**
   - 批量复制卡密：只复制卡密（mio-xxx），用于管理
   - 批量复制：复制完整卡号信息，用于支付

2. **筛选结果**
   - 默认不包括已删除的卡片（除非特别选择）
   - 自动更新过期状态
   - 支持小数额度（如 1.5）
   - 可以组合多个筛选条件

3. **权限要求**
   - 需要登录才能使用新功能
   - 访客模式不支持

## 🔍 常见场景

### 场景1：批量备份卡密
```
卡片列表 → 全选 → 批量复制卡密 → 粘贴到文本文件
```

### 场景2：查找特定额度卡片
```
卡片列表 → 输入额度 → 搜索 → 查看结果
```

### 场景3：批量管理相同额度
```
卡片列表 → 输入额度 + 选择状态 → 搜索 → 
选中卡片 → 批量操作（复制/激活/标记等）
```

### 场景4：精确筛选
```
卡片列表 → 额度:1 + 状态:已激活 + 使用:未使用 → 
搜索 → 批量复制卡密
```
